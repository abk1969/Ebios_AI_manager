**📚 CONTENU THÉORIQUE (20 MINUTES) :**

**🌐 1. Cybercriminels spécialisés santé**
```
GROUPES RANSOMWARE HOSPITALIERS :
✅ Ryuk/Conti (inactifs mais successeurs actifs) :
   • Spécialisation : Hôpitaux et collectivités
   • Tactiques : Phishing ciblé + mouvement latéral
   • Rançons : 500k€-5M€ selon taille établissement
   • Exemples : CHU Rouen (2019), Hôpitaux irlandais (2021)

✅ LockBit/BlackCat (actifs 2024) :
   • Spécialisation : Infrastructures critiques santé
   • Tactiques : RaaS + double extorsion
   • Rançons : 1M€-10M€ + menace publication données
   • Exemples : Hôpitaux US (2023), CHU européens (2024)

✅ Nouveaux groupes émergents :
   • Spécialisation croissante secteur santé
   • Tactiques évoluées (IA, deepfakes)
   • Monétisation diversifiée (crypto, NFT)
   • Géolocalisation : Russie, Corée du Nord, Iran

TRAFIQUANTS DONNÉES MÉDICALES :
✅ Marché noir données santé :
   • Prix : 50€-1000€/dossier selon complétude
   • Demande : Fraude assurance, usurpation identité
   • Plateformes : Dark web, forums spécialisés
   • Volume : Millions de dossiers échangés/an

✅ Techniques d'exfiltration :
   • Accès privilégié corrompu (personnel interne)
   • Exploitation vulnérabilités systèmes médicaux
   • Ingénierie sociale ciblée (personnel soignant)
   • Attaques chaîne d'approvisionnement (fournisseurs)
```

**🌍 2. Menaces étatiques et géopolitiques**
```
ESPIONNAGE RECHERCHE MÉDICALE :
✅ Chine (APT1, APT40) :
   • Cibles : Recherche vaccins, traitements innovants
   • Méthodes : Spear phishing, supply chain attacks
   • Objectifs : Transfert technologique, avantage économique
   • Exemples : Vol recherche COVID-19 (2020-2021)

✅ Russie (APT28, APT29) :
   • Cibles : Infrastructures santé occidentales
   • Méthodes : Malware sophistiqué, living off the land
   • Objectifs : Déstabilisation, préparation cyberguerre
   • Exemples : Attaques hôpitaux ukrainiens (2022-2024)

✅ Corée du Nord (Lazarus) :
   • Cibles : Laboratoires pharmaceutiques, recherche
   • Méthodes : Ransomware, vol cryptomonnaies
   • Objectifs : Financement programme nucléaire
   • Exemples : WannaCry impact hôpitaux (2017)

DÉSTABILISATION SYSTÈMES SANTÉ :
✅ Objectifs stratégiques :
   • Affaiblissement moral populations
   • Test résilience infrastructures critiques
   • Démonstration capacités cyber
   • Préparation conflits futurs

✅ Techniques privilégiées :
   • Attaques coordonnées multi-sites
   • Compromission long terme (APT)
   • Sabotage équipements médicaux
   • Désinformation santé publique
```

### **🏥 MODULE 2.2 - ANALYSE DES MENACES INTERNES (25 MINUTES)**

**🎯 OBJECTIFS D'APPRENTISSAGE :**
- Identifier les menaces internes spécifiques au milieu hospitalier
- Analyser les facteurs de risque du personnel médical et technique
- Évaluer l'impact des prestataires et visiteurs
- Définir des profils de risque par catégorie de personnel

**📚 CONTENU THÉORIQUE (18 MINUTES) :**

**👨‍⚕️ 1. Personnel médical et soignant**
```
MÉDECINS ET SPÉCIALISTES :
✅ Profil de risque :
   • Accès privilégié : Tous dossiers patients de spécialité
   • Mobilité : Consultations multi-sites, télémédecine
   • Autonomie : Décisions médicales, prescriptions
   • Pression : Charge travail, responsabilité vitale

✅ Motivations potentielles :
   • Financière : Revente données, corruption laboratoires
   • Personnelle : Vengeance, reconnaissance, négligence
   • Idéologique : Protestation système santé
   • Contrainte : Chantage, menaces familiales

✅ Capacités d'action :
   • Accès légitime données sensibles
   • Connaissance organisation interne
   • Confiance patients et collègues
   • Mobilité inter-établissements

PERSONNEL INFIRMIER :
✅ Profil de risque :
   • Accès étendu : Dossiers patients, prescriptions
   • Proximité : Contact direct patients 24h/24
   • Stress : Surcharge travail, conditions difficiles
   • Rotation : Équipes multiples, intérimaires

✅ Vulnérabilités spécifiques :
   • Formation sécurité insuffisante
   • Partage comptes (urgences)
   • Négligence procédures (urgence vitale)
   • Épuisement professionnel (burnout)

PERSONNEL TECHNIQUE ET SUPPORT :
✅ Administrateurs IT :
   • Accès technique complet systèmes
   • Connaissance vulnérabilités
   • Pression maintenance 24h/24
   • Sous-effectifs chroniques

✅ Personnel biomédical :
   • Accès équipements critiques
   • Maintenance préventive/curative
   • Connaissance techniques spécialisées
   • Interventions urgentes non planifiées
```

**🏭 2. Prestataires et tiers**
```
PRESTATAIRES MAINTENANCE :
✅ Risques spécifiques :
   • Accès physique non escorté
   • Interventions hors heures ouvrées
   • Équipements de test (clés USB, laptops)
   • Rotation personnel importante

✅ Vecteurs d'attaque :
   • Installation malware via équipements
   • Vol informations sensibles
   • Sabotage équipements critiques
   • Ingénierie sociale personnel

PRESTATAIRES NETTOYAGE :
✅ Exposition aux risques :
   • Accès locaux sensibles (serveurs, bureaux)
   • Horaires décalés (surveillance réduite)
   • Formation sécurité minimale
   • Turnover personnel élevé

✅ Incidents typiques :
   • Vol équipements (laptops, disques)
   • Photographie documents sensibles
   • Installation dispositifs d'écoute
   • Ouverture accès non autorisés

ÉTUDIANTS ET STAGIAIRES :
✅ Caractéristiques de risque :
   • Accès temporaire mais étendu
   • Formation sécurité incomplète
   • Curiosité naturelle (apprentissage)
   • Supervision variable selon service

✅ Scénarios de compromission :
   • Exploration non autorisée systèmes
   • Partage involontaire informations
   • Utilisation personnelle ressources
   • Négligence procédures sécurité
```
